__LINE__ --line number

"__LINE__" is expanded by preprocessor to the line number of the line where "__LINE__" is written in the file where the source code is written. 
__LINE__

Sample to get line number 

#include 

int main (void) {
  printf("%d\n", __LINE__);
}
Output result.
