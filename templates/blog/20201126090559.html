int16_t --Signed 16bit integer type

"Int16_t" is a signed 16-bit integer type. It can be used by including the " stdint.h" header. This is the type introduced in C99. 
# Signed 16bit integer type
int16_t
The maximum value of an integer that can be represented by a signed 16-bit integer type is "32767", and the minimum value is "-32768". The maximum value is " INT16_MAX" and the minimum value is " INT16_MIN". It is defined.
