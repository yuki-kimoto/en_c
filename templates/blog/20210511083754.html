    stderr-a pointer to a FILE structure that means standard error stream stderr is a pointer to the FILE structure, which means standard error stream. The standard error stream is automatically opened at the start of the program. Standard error stream generally means the output of errors to the screen. You can use it by reading the stdio.h header. 
    #include 

stderr


stderr is implemented as an expression in the C language specification. stderr can be implemented as a  global variable  or a  macro .


     Sample using stderr 

This is a sample using stderr. The  fprintf function  specifies the standard error as the output destination.


    #include 

int main (void) {
  const char * message = "Hello";
  fprintf (stderr, "% s \ n", message);
}


This is the execution result. "Hello" is displayed on the screen.
