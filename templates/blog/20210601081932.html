putchar関数 - 1文字出力する

putchar関数で、1文字出力できます。stdio.hヘッダを読み込むと使えます。 
#include 
int putchar(int c);

putchar関数のサンプル

putchar関数で1文字づつ出力するサンプルです。 
#include 
#include 
#include 

int main(void) {
  const char* message = "Hello";

  int32_t message_length = strlen(message);

  for (int32_t i = 0; i < message_length; i++) {
    putchar(message[i]);
  }
  putchar('\n');
}
